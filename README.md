# InterviewAce AI

InterviewAce AI is an intelligent interview preparation platform that helps users practice interviews for different job roles through natural conversations. The application generates role-specific interview questions using Gemini AI, asks context-aware follow-up questions based on user responses, and simulates a realistic interview experience.

## Features

* Role-based interview generation
* Technical and HR interview modes
* AI-generated follow-up questions
* Multi-turn conversational interviews
* Real-time chat interface
* React + Tailwind CSS frontend
* Express.js backend
* Gemini AI integration
* Extensible architecture for voice input/output and interview feedback

## Tech Stack

### Frontend

* React.js
* JavaScript
* Tailwind CSS
* Axios
* React Context API

### Backend

* Node.js
* Express.js
* REST APIs
* dotenv
* CORS

### AI

* Google Gemini API

## Project Objective

The goal of InterviewAce AI is to provide an interactive AI Interview Practice Partner that helps candidates improve communication, confidence, and interview performance through realistic mock interview sessions.
